# 🌍 GDP Dashboard - Cloud Lab Project

## 📌 Description
This project is an interactive web application built using Streamlit to visualize GDP (Gross Domestic Product) data of countries over time. Users can filter by country and year range to analyze economic growth trends.

## 🚀 Live Application
https://gdp-dashboard-ev59jr1zzw.streamlit.app

## 🛠️ Technologies Used
- Python
- Streamlit
- Pandas
- GitHub
- Streamlit Community Cloud

## ✨ Features
- 📈 Interactive GDP line chart
- 🌎 Country selection using names (not codes)
- 📅 Year range filtering
- 📊 GDP comparison metrics
- ⚡ Fast performance using caching
- ☁️ Live deployment with auto updates
